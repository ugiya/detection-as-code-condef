# Safety & scope boundaries

- **Original work only.** This repo contains original detections and notes. It does **not** reproduce, summarize, or host any commercial courseware, proprietary lab books, or vendor content. Technique knowledge is sourced from public [MITRE ATT&CK](https://attack.mitre.org) and open tooling documentation.
- **Isolated, deliberately-vulnerable lab.** Detonations run in a self-contained range with snapshot/revert. Attacks are never run against systems outside that private lab.
- **No secrets published.** Lab IP addresses, credentials, and remote-access details are kept out of this repository. Detections are written against standard Windows/Splunk field names so they stay portable and safe to publish.
- **Defensive intent.** The offensive step exists only to generate labeled telemetry for building and validating detections.
